El proyecto incluye la integración de un nuevo algoritmo en el sistema principal. Este desarrollo fue realizado utilizando software de código abierto y herramientas de configuración avanzadas. El módulo optimizador se enfoca en reducir el tiempo de procesamiento.

Los datos iniciales fueron almacenados en un servidor seguro, mientras que las actualizaciones se sincronizan automáticamente. Además, el programa incluye un sistema de validación que garantiza la integridad del hardware. Las pruebas finales del software se realizarán en noviembre de 2023.
